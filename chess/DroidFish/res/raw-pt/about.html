Sobre

Chess Ultimate é uma característica-rica interface gráfica de usuário de xadrez, combinado com o motor de xadrez muito forte.

Características

  • Abrindo livro
  • Desfazer / refazer ilimitado
  • Copiar / colar de / para área de transferência
  • Modo de análise
  • Modo de dois jogadores
  • Computador vs modo computador
  • Posições de instalação
  • Apoio PGN extensa
  • Força de jogo ajustável
  • Terceiros motores UCI
  • Gaviota endgame tablebases

Dicas

  • No modo de jogo, tabuleiro de xadrez longo de imprensa para ativar copy / paste do menu.
  • Em modo de edição bordo, tabuleiro de xadrez longo de imprensa para ativar o menu de edição.
  • Deslocamento para a esquerda / direita no tabuleiro de xadrez para desfazer / refazer jogadas.
  • Se o jogo contém múltiplas variações, vá para cima / para baixo no tabuleiro de xadrez para ir para a variação anterior / seguinte.
  • Longas prima Mode / esquerda / direita botões para comandos adicionais.
  • Quando voltar a jogar um jogo, alterar o modo de jogo para editar / re-jogo para ser capaz de desfazer / refazer movimentos solteiras.

Livros de Abertura

Xadrez final contém um livro pequena abertura interna. Se você quiser usar um grande livro, você pode configurar Xadrez usar poliglota externo CTG ou arquivos de livro:

  1. Copiar um ou mais poliglotas arquivos de livro para o diretório de Xadrez final no cartão SD. Livros Polyglot deve ter a extensão de arquivo. Bin. Atenção! O sistema de arquivos do Android é case sensitive, então a extensão deve ser. Bin, não.Bin ou./bin
  2. Copiar um ou mais arquivos de livro CTG para o diretório Chess Ultimate. Um livro CTG é composto por três arquivos com extensões. Ctg. Ctb e. Cto. Você deve copiar todos os três arquivos.
  3. No menu do programa, ativar a função de livro Select abertura.
  4. Selecione o arquivo do livro de abertura que você deseja usar.

Arquivos PGN

Chess Ultimate pode carregar jogos de arquivos PGN. Para usar este recurso:

  1. Copiar um ou mais arquivos PGN para o diretório Xadrez / pgn no cartão SD.
  2. Mantenha pressionado o tabuleiro de xadrez e selecione Carregar jogo a partir do arquivo PGN.
  3. Selecione o arquivo, então o jogo dentro do arquivo que deseja carregar.

Para editar cabeçalhos de jogo, como os nomes dos jogadores, prima o movimento lista e selecione Editar cabeçalhos. Para editar comentários para um movimento longo, pressione a lista de movimentos e selecione Editar Comentários.

Para salvar o seu jogo editado, prima o tabuleiro de xadrez e selecione Salvar jogo paraArquivo PGN. Selecione o arquivo para salvar, em seguida, selecione a posição no arquivo onde o novo jogo será salvo.

Atenção! Se o OI File Manager está instalado, qualquer diretório pode ser usado para carregar / salvar jogos.

Para adicionar análise de computador para a árvore de jogo, prima o análise de informações quando o computador está pensando e selecione Adicionar Análise. A principal variação de corrente é adicionado como uma variação na árvore de jogo.

No modo de análise, é possível adicionar move nulos para a árvore do jogo. Esteja ciente de que que se move nulos não são incluídos na norma PGN e pode não funcionar em PGN outro leitura de programas de xadrez.

Arquivos scid

Se você tiver Scid em movimento instalado, você pode ler arquivos de banco de dados scid diretamente de Xadrez. Mantenha pressionado o tabuleiro de xadrez e selecione Carregar jogo a partir do arquivo Scid.

Motores da UCI

Chess Ultimate pode usar motores de terceiros partidos da UCI. Para usar este recurso:

  1. Copiar um ou mais motores UCI binários para o Xadrez / uci diretório no cartão SD. Atenção! Os binários devem ser compilados para o Android.
  2. No menu, escolha Selecionar Mecanismo de Xadrez e selecionar o motor a ser usado.

Gaviota endgame tablebases

Para usar Gaviota endgame tablebases :

  1. Copiar. Gtb.cp4 arquivos para o diretório Xadrez / GTB no cartão SD.
  2. Use o Endgame Tablebases tela de preferências para controlar como os tablebases são usados.
