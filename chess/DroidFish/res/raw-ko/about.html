개장 도서

궁극의 체스는 작은 내부 오프닝 도서가 포함되어 있습니다.당신이 사용하고 싶다면큰 책은, 당신은 외부 수개 국어로 쓴을 사용하는 체스를 구성할 수 있습니다 또는 도서 파일을 CTG :

  1. 복사 하나 이상의 수개 국어로 쓴 책 파일을 SD 카드에 대한 궁극적인 체스 디렉토리에.수개 국어로 쓴 서적은 파일 확장자를 사용해야합니다. bin을. 참고: 안드로이드 파일 시스템이 대소문자를 구분하므로 확장자가. 빈 없어야합니다.Bin or .BIN [ ]
  2. 복사 하나 이상의 CTG 번호부 파일 궁극 체스 디렉토리에.CTG 책은 파일 확장명을 가진 세 파일 구성되어 있습니다. ctg. ctb합니다. 기술 책임자 (CTO).당신은 모든 세 파일을 복사해야합니다.
  3. 프로그램 메뉴에서 선택 오프닝 도서 기능을 활성화합니다.
  4. 사용하려는 여는 책 파일을 선택합니다.

PGN 파일

궁극의 체스는 PGN 파일에서 게임을 로드할 수 있습니다.이 기능을 사용하려면 :

  1. 복사 하나 이상의 PGN 파일을 SD 카드에 체스 / pgn 디렉토리.
  2. 긴 체스 보드를 누르면 PGN 파일에서로드 게임을 선택합니다.
  3. 당신로드하고자하는 파일 내의 게임 다음 파일을 선택합니다.

이러한 플레이어의 이름과 같은 게임 헤더를 편집하려면 긴 이동 목록을 눌러 선택 헤더를 편집합니다.움직임에 대한 코멘트를 편집하려면 긴 이동 목록을 눌러댓글을 수정을 선택합니다.

수정한 게임을 저장하려면, 장기 체스 보드를 눌러 게임에 저장을 선택PGN 파일.에 저장하려면 파일을 선택한 후 파일의 위치를​​ 선택하십시오어디 새로운 게임이 저장됩니다.

참고: 경우 OI File Manager 어떤 디렉토리가로드 / 게임을 저장하는 데 사용할 수있는, 설치됩니다.

게임 트리의 컴퓨터 분석을 추가하려면 오랫동안 컴퓨터를 생각하면 분석 정보를 눌러 추가 분석을 선택합니다.현재의 주요한 변화는 게임 트리의 변동으로 추가됩니다.

분석 모드에서는 게임 트리에 null이 동작을 추가할 수 있습니다.하지만 명심NULL 움직임은 PGN 표준에 포함되어 있지 않고, 다른 PGN으로 작동하지 않을 수체스 프로그램을 읽고.

Scid 파일

당신이있다면 어디서나 Scid 설치, 당신은 체스에서 직접 Scid 데이터베이스 파일을 읽을 수 있습니다.긴 체스 보드를 누르면 Scid 파일에서로드 게임을 선택합니다.

UCI 엔진

궁극의 체스는 타사 UCI 엔진을 사용할 수 있습니다.이 기능을 사용하려면 :

  1. 체스 / SD 카드에 UCI 디렉터리로 복사 하나 이상의 UCI 엔진 바이너리. 참고: 바이너리는 안드로이드에 대해 컴파일되어야합니다.
  2. 메뉴에서 체스 엔진을 선택한 선택하고 엔진이 사용하도록 선택합니다.

Gaviota 최종 목표 tablebases

사용하려면 Gaviota 최종 목표는 tablebases :

  1. 복사. gtb.cp4 파일을 SD 카드에있는 체스 / gtb 디렉토리.
  2. Use the Endgame Tablebases preferences screen to control how the tablebases are used.
