About

Окончательный Шахматы многофункциональный графический интерфейс пользователя в шахматы, в сочетании с очень сильный шахматный движок.

Свойства

  • Открытие книги
  • Неограниченный откат / повтор
  • Копирование / вставка из / в буфер обмена
  • Режим Анализа
  • Два игрока режиме
  • Компьютер против компьютера в режиме
  • Настройка позиции
  • Расширенная поддержка PGN
  • Регулируемая сила игры
  • Третья партия двигателей UCI
  • Гавиота эндшпиль tablebases

Советы

  • В режиме игры, давно шахматная доска пресс для активации копировать / вставить меню.
  • В режиме редактирования доску, долго шахматной доске пресс активировать меню редактирования.
  • Выделите влево / вправо на шахматной доске, чтобы отменить / повторить движения.
  • Если игра содержит несколько изменений, прокрутки вверх / вниз на шахматной доске для перехода к предыдущей / следующей вариации.
  • Длительное нажатие в режиме / влево / вправо для дополнительных команд.
  • При повторной игрой, изменить режим игры на Редактировать / переиграть Игру иметь возможность отмены / повтора одного движения половины.

Открытие книги

Окончательный шахматы содержит небольшое внутреннее открытие книги. Если вы хотите использовать большая книга, вы можете настроить шахматы использовать внешние полиглот или КТГ книге файлов:

  1. Копирование одного или нескольких полиглот книга файлы Окончательный каталог шахматной на карте SD. Полиглот книги должен иметь расширение файла. Ящик. Обратите внимание! Система Android файлов чувствительны к регистру, так что расширение должно быть. Бен, нет.Бен или.BIN [ ]
  2. Копирование одного или нескольких КТГ книга файлы Окончательный каталог шахматы. КТГ Книга состоит из трех файлов с расширениями файлов. КТГ. CTB и. КТО. Необходимо скопировать все три файла.
  3. В меню программы, активировать функцию выбора книги открытия.
  4. Выберите файл, открытие книги, которую вы хотите использовать.

PGN файлов

Окончательный шахматы можно загружать игры из PGN файлов. Для использования этой функции:

  1. Копирование одного или нескольких PGN файлов в шахматы / PGN каталог на карте памяти SD.
  2. Длительное нажатие на шахматной доске, затем Загрузить игру из файла PGN.
  3. Выберите файл, то игра в файл, который Вы хотите загрузить.

Чтобы изменить игру заголовки, такие как имена игроков, давно нажмите движение список и выберите Изменить заголовок Чтобы редактировать комментарии для перехода нажмите и удерживайте кнопку ходу списка и выберите Изменить комментарии.

Чтобы сохранить измененную игру, долго нажимать на шахматной доске и выберите Сохранить игруPGN файл. Выберите файл для сохранения, а затем выберите позицию в файле где новая игра будет сохранена.

Обратите внимание! Если OI File Manager установлен, любой каталог может быть использована для загрузки / сохранения игры.

Чтобы добавить компьютерный анализ в игру дерева, долгое нажмите анализ информации, когда компьютер думает и выберите пункт Добавить анализа. В настоящее время изменение основных добавляется изменение в игре дерево.

В режиме анализа, можно добавить нулевой движется в игру дерева. Имейте в виду что нулевой движения не входят в стандартный PGN и не могут работать в других PGN чтение шахматных программ.

SCID файлы

Если вы SCID на ходу установки, вы можете прочитать SCID файлы базы данных непосредственно из шахмат. Длительное нажатие на шахматной доске, затем Загрузить игру от SCID файл.

UCI двигателей

Окончательный шахматы можно использовать сторонние UCI двигателей. Для использования этой функции:

  1. Копирование одного или нескольких двигателей UCI файлы в шахматы / UCI каталог на карте памяти SD. Обратите внимание! Файлы должны быть скомпилированы для Android.
  2. В меню выберите пункт Выбрать движком и выбрать двигатель для использования.

Гавиота эндшпиль tablebases

Для использования Гавиота эндшпиль tablebases :

  1. Копировать. Gtb.cp4 файлы в шахматы / GTB каталог на SD-карте.
  2. Используйте Endgame Tablebases предпочтения экран для управления tablebases используются.
